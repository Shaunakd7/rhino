# Product Images Directory

Place your product images in this directory with the following naming convention:

## Required Images

- `aura.jpg` (or `.png`, `.webp`) - For AURA Deep Gloss product
- `lumo.jpg` - For LUMO Performance Gloss product
- `prism.jpg` - For PRISM Ultra Gloss product
- `dusk.jpg` - For DUSK Matte Stealth product
- `haze.jpg` - For HAZE Satin Matte product
- `k-series.jpg` - For K-Series introduction section (optional, will show gradient if not present)

## Image Recommendations

- **Format**: JPG, PNG, or WebP
- **Aspect Ratio**: 16:9 or 21:9 (wide format works best)
- **Resolution**: Minimum 1920x1080px, ideally 3840x2160px or higher
- **File Size**: Optimize images to keep file sizes reasonable (< 2MB per image)

## Image Content Suggestions

Each product image should showcase:
- The finish quality (glossy or matte)
- Automotive application context
- Premium aesthetic aligned with RHINO brand
- High-quality, cinematic visuals

The website will automatically display these images in the product showcase sections. If an image is not found, the section will gracefully fall back to gradient backgrounds.
